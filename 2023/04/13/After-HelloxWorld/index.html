hello world后第一篇文章

发表于2023-04-13|更新于2023-04-20|文档
|阅读量:

记录一下我做这个博客的过程：
1，安装nodejs和git
2，桌面新建一个文件夹
3，文件夹内cmd，npm install hexo(安装hexo)
4，npx hexo init blog（初始化hexo到blog文件夹）
5，切换到bolg文件夹
6，npm install

【如果出现报错如下：
4 vulnerabilities (2 high, 2 critical)

To address all issues (including breaking changes), run:
npm audit fix –force)

按提示运行npm audit fix –force解决

如果没有报错请忽略】
7，npx hexo g（生成网页）npx hexo s(本地服务器运行预览网页)
8，修改配置文件（参考hexo官网文档提示修改）
9，npm install hexo-deployer-git –save（）
9，git config –global user.email “你的邮箱” git config –global user.name “你的昵称”（这里好像不使用gitee的邮箱和名称也可以）
10，npx hexo d（运行命令部署到gitee,gitee地址在配置文件中已设置）

大致如上
